# Install

## macOS Notes

If you use MelonLoader on macOS, install through the modlist.org app. After installation, copy the native launch options from the app and paste them into your Steam launch arguments.

## Quick Install

1. Download the latest Quartz release from [GitHub releases](https://github.com/QuartzTeam/Quartz/releases/latest).
2. Choose `Quartz.zip` for MelonLoader or `QuartzUmm.zip` for UnityModManager.
3. Install the zip with your loader.
4. Start A Dance of Fire and Ice and open the Quartz menu in game.

## Updating

Download the newest release and replace the previous Quartz zip in your loader. If the game keeps loading the old version, restart Steam and confirm only one Quartz package is installed.
