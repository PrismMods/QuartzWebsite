# Troubleshooting

### Quartz does not appear

- Confirm the zip matches your loader.
- Confirm the loader is installed for A Dance of Fire and Ice.
- Check that the game was restarted after installing Quartz.

### The game starts but settings are missing

Open the in-game Quartz menu. For UMM installs, Quartz settings live in the Quartz menu rather than inside the UMM panel.

### A download link is blocked

Use the latest release page directly:

```text
https://github.com/QuartzTeam/Quartz/releases/latest
```
