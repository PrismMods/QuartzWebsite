# Quartz Documentation

Quartz is an all-in-one mod for A Dance of Fire and Ice. This documentation is split across regular markdown files, so adding a page is just adding another `.md` file and listing it in `manifest.json`.

## Pages

- [Install](?page=install)
- [Troubleshooting](?page=troubleshooting)

## Loader Choice

- **MelonLoader** is the recommended install path for most players.
- **UnityModManager** is useful if your ADoFaI setup already depends on UMM.
- Both packages expose the same Quartz menu and settings once loaded.
