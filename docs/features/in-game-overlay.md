# In-Game Overlay Fonts

Applies the font you've picked for Quartz's own UI to a few pieces of A Dance of Fire and Ice's native HUD text as well, so your chosen font shows up during play, not just in the mod's menu.

## Where to find it

Open the Quartz menu in-game, go to the **Settings** tab, open the **Font** section, and look for **In-Game Overlay**.

## Highlights

- **Song Title** — the level title shown during play.
- **Countdown** — the pre-level "3, 2, 1, Go!" countdown.
- **Judgement** — the per-hit rating popup ("Perfect!"/"Early!").
- Each toggle is independent and off by default; turning one on uses whichever font is selected above in the **Font** section.

!!! note
    The **Judgement** toggle here only re-fonts the per-hit rating popup. It's separate from the [Judgement](judgement.md) overlay, which shows accumulated hit counts.
